# 🌑 DadGPT Dark Web Search Module

This directory contains documentation and tools for DadGPT's built-in darkweb search capabilities.

## 🔍 What is Dark Web Search?

Dark web search allows accessing .onion sites and deep web content that traditional search engines cannot index.

### Features
- 🔎 Search .onion domains
- 🌐 Access hidden services
- 🔒 Privacy-preserving search
- 🚫 No censorship or content filtering
- 📊 Index uncensored information

### Supported Search Engines
- Ahmia (https://ahmia.fi)
- Torch (http://xmh57jrknzkhv6y.onion)
- Not Evil (http://hss3uro2hsxfogfq.onion)
- Haystak (http://haystakvxad7wbm5.onion)
- DarkSearch (http://darksearchfi7lj.onion)

## 🚀 Usage

Visit [https://dadgpt.live/darkweb](https://dadgpt.live/darkweb) to use the darkweb search feature directly through DadGPT.

## 🔐 Privacy Notes

- All searches are anonymous
- No logs kept
- No tracking
- Uses Tor network for anonymity

## ⚠️ Disclaimer

Use darkweb search responsibly. DadGPT is not responsible for content accessed through darkweb search.
